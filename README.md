# Pokédex
A "Pokédex" using api [PokeApi](https://pokeapi.co/).

## Requirements (interface)
Type the following commands into the terminal:

```
pip install PySimpleGUI
pip install requests
```
## Requirements (no interface)
Type the following commands into the terminal:

```
pip install requests
```
